# SchemaFX Application Definition

This repository outlines the standard to SchemaFX's schema structure. This structure aims to create a standard to define data-driven applications.

The schema defines what an application is, while interpreters are responsible for defining how it runs. The schema by itself is purposefully limited and declarative.

## Contents

- `spec/` — The authoritative specification
- `examples/` — Complete sample schemas
